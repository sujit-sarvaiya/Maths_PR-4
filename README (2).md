# 📊 Probability Distributions & Data Analysis Project

## 📌 Overview

This project focuses on analyzing customer transaction data using
statistical distributions and transformations. It helps in understanding
data behavior and deriving meaningful insights.

------------------------------------------------------------------------

## 🎯 Objective

-   Understand probability distributions
-   Analyze real-world transaction data
-   Apply statistical models and transformations
-   Visualize and interpret data patterns

------------------------------------------------------------------------

## 🧾 Dataset Description

The dataset includes: - transaction_id - customer_id -
transaction_amount - transaction_date - transaction_count - region -
transaction_status

------------------------------------------------------------------------

## 🛠️ Tools & Libraries

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn
-   SciPy
-   Statsmodels

------------------------------------------------------------------------

## 📚 Part A -- Theory Covered

-   Statistical Distribution
-   Q-Q Plot
-   Discrete vs Continuous Data
-   Bernoulli Distribution
-   Binomial Distribution
-   Poisson Distribution
-   Log-Normal Distribution
-   Power Law Distribution
-   Box-Cox Transform
-   Z-score
-   PDF & CDF

------------------------------------------------------------------------

## 💻 Part B -- Practical Tasks

-   Data loading and preprocessing
-   Bernoulli & Binomial distribution analysis
-   Poisson distribution for daily transactions
-   Log transformation for skewed data
-   Q-Q Plot for normality check
-   Box-Cox transformation
-   Z-score calculation
-   PDF and CDF visualization

------------------------------------------------------------------------

## 📊 Key Insights

-   Transaction amounts are skewed → Log-Normal fit
-   Daily transactions → Poisson distribution
-   Success/Failure → Bernoulli distribution

------------------------------------------------------------------------
