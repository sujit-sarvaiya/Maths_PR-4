## 🧠 Concepts Explanation

### 🔹 Bernoulli Distribution
- Used for experiments with only two outcomes (Success/Failure)
- Example: Tossing a coin

### 🔹 Binomial Distribution
- Extension of Bernoulli for multiple trials
- Example: Number of heads in multiple coin tosses

### 🔹 PDF
- Shows probability of a specific value

### 🔹 CDF
- Shows cumulative probability up to a value
